# Paper Card 模板

## 模板

```markdown
paper_id: "arXiv:xxxx.xxxxx / DOI"
source_url: ""
title: ""
year: 2025
venue: ""
evidence_status: "Verified / Plausible but unverified / Not found"
one_sentence_takeaway: ""
what_it_changes_for_us:
  - "实验设计："
  - "指标/评估："
  - "写作站位："
key_results:
  - "关键结果（带条件/范围）"
limitations_and_traps:
  - "不适用前提/复现难点/混淆因素"
assets:
  code: ""
  data: ""
  benchmark: ""
citation_bibtex: "@..."
```

## 使用规则

1. paper_id必须可查证，禁止编造
2. 引用了外部来源时，source_url必须可追溯
3. evidence_status必须明确，不能把推测写成已验证事实
4. limitations_and_traps必须写，至少1条
5. what_it_changes_for_us必须落地到具体行动
